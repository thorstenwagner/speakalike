# FastSpeak - Text-to-Speech Anwendung

Eine einfache Anwendung, die Coqui TTS nutzt, um Textnachrichten vorzulesen.

## Installation

```bash
pip install -r requirements.txt
```

## Verwendung

```bash
python main.py
```

## Features

- Text-to-Speech mit Coqui TTS (deutsches Thorsten-Modell)
- Einfache grafische Benutzeroberfläche
- Einstellbare Sprechgeschwindigkeit
- Mehrsprachige Unterstützung
- Fallback auf pyttsx3 bei fehlender Coqui-Installation

## Anforderungen

- Python 3.8+
- Windows/Linux/macOS
